Reference Guide - HMA Security Token Service version 2.5

Axis2 Documentation

The security token service is based on the Axis2 webservice framework. The documentation of Axis2 can be found from http://ws.apache.org/axis2/1_4/contents.html

Tomcat Documentation

Tomcat server is used by the security token service (Axis2 based service). More information about Tomcat server, such as user guide, administration guide documentation can be found from  http://tomcat.apache.org/tomcat-6.3-doc/index.html
